01 / THE CONFLICT

The Problem

Aircraft and missiles send telemetry — altitude, speed, position, sensor data — back to the ground as radio signals. The frequency bands they use now overlap with 5G and LTE networks. When a cell tower transmits over the same band, the interference can be stronger than the telemetry signal itself, causing the signal to be completely lost.

02 / LIMITS OF DSP

Why Traditional Receivers Fail

Classical receivers were built to pull a signal out of random background static. They assume interference is weak and noise-like. But 5G/LTE isn't random noise — it's a strong, structured signal sitting right on top of yours. A traditional receiver can't tell the two apart, so it rejects or distorts both, failing to extract any telemetry.

03 / CONFORMER EXCISION

What This Does

This is a deep-learning receiver that learns what cellular interference looks like and separates it from the telemetry signal — like picking out one voice in a loud room instead of just turning the volume down. A neural network reconstructs the clean signal, and error-correction coding fixes whatever remains. The result: the original message is recovered exactly, even when the interference outpowers the signal.

Core Excision Engineering

🎯

Segmented Sync

Classic correlation is split into 4 segments of 8 µs. Combines GPS-style acquisition grids to withstand severe carrier frequency offsets up to ±240 kHz.

⚡

Spectral Excision

SpectralGate maps incoming noisy IQ spectra into standardized log-magnitude/phase domains to selectively isolate and excise wideband blockers.

🧠

Conformer Blocks

Conformer layers utilize both self-attention and depth-wise convolutions to capture long-range temporal dependencies and restore phase integrity.

🛡️

Min-Sum LDPC

Calibrated Log-Likelihood Ratios feed a rate-1/2 LDPC decoder (50 max iterations), restoring corrupted signals error-free even during severe jammer overlaps.

Technical Glossary

Deep dive into the signals, networks, and DSP algorithms powering the receiver.

PCM-FM Modulation
Pulse Code Modulation - Frequency Modulation. The standard flight telemetry modulation format (defined in IRIG-106 Chapter 4) where bits modulate carrier frequency offsets.
Conformer (v8.5)
A neural network combining Convolution layers (learning local wave shapes/phases) and Transformers (learning long-range temporal dependencies) for advanced signal recovery.
LDPC Error Correction
Low-Density Parity-Check code. A highly efficient linear block error-correction code matching Shannon limit. Resolves residual bit flips post-Conformer to guarantee zero errors.
CFO Calibration
Carrier Frequency Offset calibration. Segmented correlation locks onto sync word components (FSW) to offset Doppler shifts up to ±240 kHz.
LLR Log-Likelihoods
The log-probability indicators for decoded bit options. Logits out of Conformer layers are temperature-calibrated before min-sum parity checks.
PSL Sync Metric
Peak Sidelobe Level metric of sync word correlation. A threshold ≥ 1.5 defines a secure lock. Anything below denotes signal alignment failure.

Frequently Asked Questions

How does the model handle out-of-distribution (OOD) interference? ▾
The excision engine utilizes domain randomization during training (v17 generator). It trains on dynamic wideband OFDM, multi-tone jammers, Swept chirps, and pulsed noise. This ensures the Conformer generalizes across standard LTE and 5G interference layers without crashing.
Why is LLR calibration needed before LDPC decoding? ▾
Conformer output logits are computed using focal-loss gradients, which typical mis-calibrates probability outputs. By fitting a temperature scaling parameter T on the validation set, we restore honest, calibrated Log-Likelihood Ratios necessary for optimal min-sum LDPC checks.
Does this run locally in the browser or on the server? ▾
The signal processing and conformer inference are executed server-side (Hugging Face Spaces) in Python for numerical accuracy. Downsampled wave structures are then pushed via JSON to the browser client, which plots them using lightweight HTML5 canvas engines.
